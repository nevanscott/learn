We can shortcut the margins a little too. Take the following CSS:

```css
p {
  margin-top: 14pt;
  margin-bottom: 14pt;
}
```

That can be shortened to:

```css
p {
  margin: 14pt 0;
}
```

In the new line, `14pt` applies to the top and bottom margins, and `0` applies to the right and left.
