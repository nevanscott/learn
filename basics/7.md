---
css: |
  #body {
    font-size: 11pt;
    line-height: 14pt;
    width: 330pt;
    margin-left: 100px;
    }
  h1 {
    text-align: center;
    }
  h2 {
    text-align: center;
    }
---

<p>We can achieve the same result in a more shortened form by combining the two rules. Separating the tags by commas lets us apply the same property to more than one tag.</p>

<pre>
body {
	font-size: 11pt;
	line-height: 14pt;
	width: 330pt;
	margin-left: 100px;
	}
<del>h1 {
	text-align: center;
	}
h2 {
	text-align: center;
	}</del>
<ins>h1, h2 {
	text-align: center;
	}</ins>
</pre>
