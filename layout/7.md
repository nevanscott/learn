    }

<p>We can force the yellow block to break below the <code>float</code>ed image by assigning it the appropriate <code>clear</code> value:</p>
<pre>
#header {
	background: #F66; /* red */
	}
#preface {
	background: #66F; /* blue */
	}
#preface img {
	float: left;
	}
#content {
	background: #FF6; /* yellow */
	<ins>clear: left;</ins>
	}
</pre>
